# ProHealth — Admin Panel

This is the admin dashboard for the ProHealth Telemedicine Platform.

📖 Full documentation, features, and setup guide:
👉 [View Main README](https://github.com/TahaY291/telemedicine-frontend)


## Environment Variables

Create a `.env` file in the root of the project and add the following:

```env
VITE_BACKEND_URL=your backend url
```

## Quick Start
git clone https://github.com/TahaY291/telemedicine-admin
cd telemedicine-admin
npm install
npm run dev
